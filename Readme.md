# ESP-NOW playground

The code is based on [this](https://randomnerdtutorials.com/esp-now-esp8266-nodemcu-arduino-ide/) article.

## Init the pio project

```console
pio boards nodemcu
```

```console
pio project init --board nodemcuv2
```

To monitor the device:

```console
pio device monitor -b 115200
```
